# チェックインとチェックアウト {#check-in-check-out}

Adobe アセットを使用すると、作業中のクリエイティブアプリから直接 AEM Assets をチェックアウトして、すぐに編集を開始できます。Creative Cloud アプリでの編集が完了したら、適切なバージョン管理やコメントを使用して、AEM Assets にアセットを簡単にチェックインすることができます。
