# AEM Assets コンテンツハブの設定

Adobe Cloud Manager プログラムとアセットアドオンを使用して Experience Manager Assets コンテンツハブを設定する方法について説明します。また、製品プロファイルを使用してユーザーのアクセスを設定する方法についても説明します。

## AEM Assets Dynamic Media OpenAPI の設定

AEM Assets Dynamic Media OpenAPI のライセンスがある場合、それらを使用して、スマート切り抜きなどのContent Hubで動的レンディションを生成することができます。 これらの Dynamic Media OpenAPI レンディションをContent Hubに表示するには、まず初期設定が必要です。
