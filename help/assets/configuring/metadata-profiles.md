# メタデータプロファイル

メタデータプロファイルを使用すると、初期設定のメタデータをアセットフォルダ内のアセットに自動的に適用できるので、AEMユーザーに対するメタデータ管理の負荷を軽減し、メタデータの一貫性を向上できます。
