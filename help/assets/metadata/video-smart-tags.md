# ビデオスマートタグ

Experience Manager Assetsは、Adobe Senseiを活用してビデオの主要な要素を説明するキーワードでビデオアセットをインテリジェントにタグ付けし、AEMユーザーによるキーワード検索で簡単に見つけることができます。
