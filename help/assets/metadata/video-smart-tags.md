# ビデオスマートタグ

Experience Managerアセットでは、Adobe Senseiを利用してビデオの主要な要素を説明するキーワードでビデオアセットにインテリジェントにタグ付けし、AEMユーザーによるキーワード検索で簡単にビデオアセットを見つけることができます。
