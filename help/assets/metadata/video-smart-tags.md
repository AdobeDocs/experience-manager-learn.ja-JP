# ビデオスマートタグ

Experience Manager Assets は Adobe Sensei を利用して、ビデオの主要な要素を説明するキーワードでインテリジェントにビデオアセットにタグを付け、AEM ユーザーによるキーワード検索で簡単に見つけられるようにします。
