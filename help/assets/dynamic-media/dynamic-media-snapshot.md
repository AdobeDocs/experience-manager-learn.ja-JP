# Dynamic Media Snapshot {#snapshot}

Dynamic Media Snapshot は、Dynamic Mediaの機能を示す視覚的なデモツールで、最適化された動的な画像配信を実現します。 テスト画像やDynamic Media URL を試して、様々なDynamic Media画像修飾子の出力を視覚的に観察し、スマートイメージングを（WebP および AVIF 配信を使用した）ファイルサイズ、ネットワーク、デバイスピクセル比 (DPR) の最適化を実施します。
