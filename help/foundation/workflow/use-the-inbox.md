# AEMでのインボックスの使用

AEM Inboxは、タスクとワークフローに関するコラボレーションと可視性を強化します。 インボックスのカレンダー表示は、プロジェクトとタスクを時間ベースの表示で追跡し、プロジェクトの管理とスケジュールを容易にします。 カレンダー表示を使用すると、現在進行中、予定中、期限切れ、期限切れのプロジェクトやタスクを全体的に表示できます。
