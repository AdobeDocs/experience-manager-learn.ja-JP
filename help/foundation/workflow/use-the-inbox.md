# AEM でのインボックスの使用

AEM のインボックスは、タスクとワークフローに関するより優れた共同作業と表示を促進します。 インボックスのカレンダー表示では、プロジェクトとタスクを時間に基づく表示で追跡できるため、プロジェクト管理とスケジュール設定が容易になります。 カレンダー表示では、現在進行中のプロジェクトやタスク、計画済みのタスク、まもなく期限切れのタスク、期限切れとなったタスクを全体的に確認できます。
