# AEMでのインボックスの使用

AEM Inboxは、タスクとワークフローに関するより優れたコラボレーションと可視性を促進します。 インボックスのカレンダー表示では、プロジェクトとタスクを時間に基づく表示で追跡し、プロジェクトの管理とスケジュール設定を容易にします。 カレンダー表示では、現在進行中のプロジェクトやタスク、計画済みのタスク、まもなく期限切れのタスクを全体的に確認できます。
