# AdobeCloud Manager の使用 — プログラム

Cloud Manager **プログラム** は、通常、購入したサービス契約 (SLA) に対応する、ビジネスイニシアチブの論理的なセットをサポートするAEM環境のセットを表します。 例えば、あるプログラムが、グローバルなパブリック Web サイトをサポートする AEM リソースを表す一方で、別のプログラムは社内の中核的 DAM を表します。
