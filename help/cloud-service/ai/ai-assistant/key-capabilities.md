# AI アシスタントの主な機能

AI アシスタントが、即座のガイダンス、コードスニペット、トラブルシューティング手順を提供することで、AEM ワークフローを高速化する方法を説明します。 これにより、サポート作業が軽減され、チームがより迅速かつ高品質にコンテンツを配信できるようになります。
