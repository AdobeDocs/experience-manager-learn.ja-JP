# EXPERIENCE MANAGERのAI

![Experience ManagerのAI](./assets/overview/hero.png)

Experience Manager as a Cloud Serviceは、コンテンツ管理の強化、ワークフローの合理化、ユーザーエクスペリエンスの向上を実現する高度なAI機能を提供します。 AEM AI アシスタントやAEM AI AgentsなどのAIを利用した機能を統合することで、タスクを自動化し、インサイトを獲得し、コンテンツ配信を最適化することができます。
