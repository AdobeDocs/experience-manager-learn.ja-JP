# テーマの作成

テーマにはコンポーネントやパネルのスタイル設定の詳細が含まれ、さまざまなフォームにテーマを再利用できます。スタイルには、背景色、状態色、透明度、配置、およびサイズなどのプロパティが含まれます。テーマをフォームに適用すると、指定されたスタイルがフォームの対応コンポーネントに反映されます。
