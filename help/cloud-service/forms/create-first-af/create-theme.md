# テーマの作成

テーマには、コンポーネントやパネルのスタイル設定の詳細が含まれており、異なるフォームでテーマを再利用できます。 スタイルには、背景カラー、ステートカラー、透明度、配置、サイズなどのプロパティが含まれます。テーマをフォームに適用すると、指定したスタイルがフォームの対応するコンポーネントに反映されます。
