# AEM FormsとAdobe Experience Platformの統合（AEP）

>[!NOTE]
>
>アダプティブForms（AEM Forms）とAdobe Experience Platform（AEP）を連携させる機能は、アーリーアクセスプログラムに含まれています。アクセスをリクエストするには、住所からaem-forms-ea@adobe.comにメールを送信してください。

AEM Forms を Adobe Experience Platform に接続して、フォームエクスペリエンスを変換できます。この強力な統合機能により、組織は、パーソナライズされたフォームエクスペリエンスにリアルタイム顧客プロファイルを活用したり、AdobeへのAEM Forms データ送信を効率化したり、Experience Platform エコシステム全体で統合された顧客レコードを作成したりできます。

このチュートリアルでは、AEM FormsとAEPの統合を作成する手順について説明します
