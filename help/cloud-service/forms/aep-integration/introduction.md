# AEM FormsとAdobe Experience Platformの統合（AEP）

>[!NOTE]
>
>アダプティブ Forms（AEM Forms）とAdobe Experience Platform（AEP）を連携する機能は、早期アクセスプログラムの下にあります。作業アドレスからaem-forms-ea@adobe.comにメールを送信してアクセスをリクエストしてください。

AEM Forms を Adobe Experience Platform に接続して、フォームエクスペリエンスを変換できます。この強力な統合により、企業はリアルタイムの顧客プロファイルを活用して、パーソナライズされたフォームエクスペリエンスを実現できます。また、Experience PlatformへのAEM Formsデータ送信を効率化し、Adobeのエコシステム全体で統合された顧客記録を構築できます。

このチュートリアルでは、AEM FormsとAEPの統合を作成する手順について説明します
