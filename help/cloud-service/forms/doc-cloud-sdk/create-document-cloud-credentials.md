# Adobe I/Oプロジェクト

1. 統合を作成する最初の手順は、 [Adobe ID](https://account.adobe.com/)
1. 次に移動： [Adobe Developer Console](https://console.adobe.io/home)
1. 手順 1 で作成したAdobe IDを使用してログインします。
1. 新しいプロジェクトを作成

ビデオに従ってAdobe I/Oプロジェクトを作成してください。
