# Adobe I/Oプロジェクト

1. 統合を作成する最初の手順は、[Adobe ID](https://account.adobe.com/)を作成することです
1. [Adobe開発者コンソール](https://console.adobe.io/home)に移動します。
1. 手順1で作成したAdobe IDを使用してログインします。
1. 新しいプロジェクトの作成

ビデオに従って、Adobe I/Oプロジェクトを作成します。
