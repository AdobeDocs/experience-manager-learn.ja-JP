# Adobe I/O プロジェクトの作成

1. 統合を作成するための最初の手順は、[Adobe ID](https://account.adobe.com/) を作成することです
1. [Adobe Developer Console](https://console.adobe.io/home) に移動します 
1. 手順 1 で作成した Adobe ID を使用してログインします
1. 新しいプロジェクトを作成します

ビデオに従って Adobe I/O プロジェクトを作成します。
