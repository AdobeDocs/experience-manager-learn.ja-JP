# バッチデータの設定

バッチデータ設定は、バッチ API 用の BLOB ストレージの特定のインスタンスを設定する場合に役立ちます。 顧客が所有する Microsoft Azure BLOB ストレージ内で入力および出力の場所を指定できます。
