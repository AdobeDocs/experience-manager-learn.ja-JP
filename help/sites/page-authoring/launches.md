# ローンチ

{{edge-delivery-services-and-page-editor}}

AEM Sites のローンチでは、今後のリリースで web サイトのコンテンツを作成およびレビューすることができます。 ローンチの作成中、実稼動 web サイトは通常どおりに進化し、毎日変更することができます。
