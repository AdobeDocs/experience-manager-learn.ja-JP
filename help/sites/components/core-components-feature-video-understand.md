# コアコンポーネントについて{#understanding-core-components}

AEMコアコンポーネントは、Adobe Experience Managerで使用される標準セットコンポーネントです。 コアコンポーネントは、Adobeのベストプラクティスと標準を基に構築され、すべてのサイト実装に対して機能の基本セットを提供します。
