# AEM用のSSLの有効化

AEM FormsとAdobe Signを統合するための前提条件の1つは、SSLを使用するようにAEMインスタンスを設定することです。 Adobe Experience ManagerのSSL設定ウィザードを使用して、AEMで実行するHTTPSインスタンスを簡単に設定できるようにします。
次のビデオでは、ウィザードを使用してSSLを設定するために必要な手順を説明します
