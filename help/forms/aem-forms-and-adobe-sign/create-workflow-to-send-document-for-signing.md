# ワークフローを使用した署名用ドキュメントの送信

ドキュメントに署名コンポーネントを使用してAEMワークフローを作成します。 自動生成されたDoRを利用して、署名コンポーネントを使用して署名用に送信します。
このビデオでは、署名用にドキュメントを送信する効果的なワークフローを作成するために必要な様々な手順を説明します。
