# 貢献度セクションへの表の追加

この部分では、貢献度セクションに表を追加します。
詳しくは、 [印刷チャネルでのテーブルの使用](/help/forms/interactive-communications/table-in-print-channel-documents-video-use.md) 印刷チャネルドキュメントでテーブルを使用する方法の詳細については、を参照してください。

AEM Formsにログインし、 Adobe Experience Manager / Forms / Forms &amp; Documents に移動します。

401KStatementPrint フォルダを開きます。

401KStatementPrint を編集モードで開きます。

次のビデオでは、印刷チャネルドキュメントでテーブルを設定する手順について説明します。
