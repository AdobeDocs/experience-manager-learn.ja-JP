# AF テンプレートを作成

このチュートリアルの目的で、新しいアダプティブフォームテンプレートを作成します。 また、新しいページコンポーネントも作成され、このテンプレートに関連付けられました。 ページコンポーネント内の JSP は、アダプティブフォームの事前入力に使用される MySql データベースからデータを取得します。
