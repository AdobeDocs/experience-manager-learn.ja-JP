# AEMワークフローのドキュメント型変数


ドキュメント型の変数は、レコード、アダプティブフォームの添付ファイルのドキュメントを保存するために使用されます。 これらの変数は、「電子メールの送信」コンポーネントで使用して、電子メールで添付ファイルを送信できます。

このビデオでは、ドキュメント型変数を使用して生成されたDoRを保存し、そのドキュメント変数をAEMワークフローの電子メール送信手順で使用する方法を説明します。
